# LD38

A game for the Ludum Dare 38 Game Jam

Theme - A Small World

## Ideen

* 2D
* Pixelart
* Island
* Side scroller
    * Round island -> looping
* You are a pirate

### Things to do

* Wood chopping
* Fishing
* Building home
* Building tools
    * Fishing spear
    * Axe
* Collect things
* Fire
* Build ships
* Hunting
* Dock
* Community buildings
* Rum
* Storage
